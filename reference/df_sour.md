# Summarise Source records in a dataframe

Summarise Source records in a dataframe

## Usage

``` r
df_sour(x, xrefs = NULL)
```

## Arguments

- x:

  A gedcom object.

- xrefs:

  A vector of xrefs to summarise. If this is left NULL, all relevant
  xrefs will be used.

## Value

A dataframe summarising a Source record on each row.
