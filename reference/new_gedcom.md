# Create a new GEDCOM object

Create a new GEDCOM object

## Usage

``` r
new_gedcom(my_language = "en")
```

## Arguments

- my_language:

  The primary language in which data will be stored. The language code
  should adhere to BCP 47.

## Value

A minimal gedcom S7 object.

## Details

See documentation for
[`GedcomS7()`](https://jl5000.github.io/gedcomS7/reference/GedcomS7.md)
and
[`GedcomSource()`](https://jl5000.github.io/gedcomS7/reference/GedcomSource.md)
for object properties.
