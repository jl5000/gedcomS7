# Summarise Individual records in a dataframe

Summarise Individual records in a dataframe

## Usage

``` r
df_indi(x, xrefs = NULL)
```

## Arguments

- x:

  A gedcom object.

- xrefs:

  A vector of xrefs to summarise. If this is left NULL, all relevant
  xrefs will be used.

## Value

A dataframe summarising an Individual record on each row.
