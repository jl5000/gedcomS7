# Validate a date range/period

Validate a date range/period

## Usage

``` r
chk_input_dates(start_date, end_date)
```

## Arguments

- start_date:

  The start date given either as a `DateCalendar` or a GEDCOM date
  string.

- end_date:

  The end date given either as a `DateCalendar` or a GEDCOM date string.

## Value

Either a character string giving an error message, or NULL.
