# Summarise Family records in a dataframe

Summarise Family records in a dataframe

## Usage

``` r
df_fam(x, xrefs = NULL)
```

## Arguments

- x:

  A gedcom object.

- xrefs:

  A vector of xrefs to summarise. If this is left NULL, all relevant
  xrefs will be used.

## Value

A dataframe summarising a Family record on each row.
