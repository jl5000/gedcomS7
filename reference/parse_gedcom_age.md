# Convert a GEDCOM age at event into decimalised years

Convert a GEDCOM age at event into decimalised years

## Usage

``` r
parse_gedcom_age(age_string)
```

## Arguments

- age_string:

  A string describing an age at an event, e.g. "14y 3m 1w 2d".

## Value

A numeric value giving the age in years.
